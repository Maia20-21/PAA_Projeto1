### Descrição do Projeto

Este projeto consiste em um programa em C que resolve o problema da "Missão no Edifício João Calvino". 
O objetivo é encontrar a chave (`*`) em um mapa 2D, seguindo as regras de movimento. 
A solução foi desenvolvida utilizando a estratégia de **backtracking** por meio de uma função recursiva, conforme os requisitos do projeto.

### Funcionalidades Implementadas

- **Algoritmo de Backtracking:** O programa utiliza uma abordagem recursiva para explorar todas as combinações de caminhos possíveis.
- **Regras de Movimento:** Os movimentos são restritos a corredores `H` (horizontal) e `V` (vertical), e a busca termina ao encontrar a chave (`*`).
- **Prevenção de Ciclos:** O algoritmo marca as posições visitadas para evitar loops infinitos.
- **Entrada e Saída:** O programa lê as dimensões, o mapa e as coordenadas de início via entrada padrão e imprime uma das duas mensagens de saída, conforme o resultado da busca.

### Compilação e Execução

Para compilar e executar o programa, utilize os seguintes comandos no terminal:

1.  **Compilação:**
    ```sh
    gcc -o missao_joaocalvino missao_joaocalvino.c
    ```

2.  **Execução:**
    ```sh
    ./missao_joaocalvino
    ```
    Após a execução, forneça a entrada de dados conforme o formato especificado no problema.
